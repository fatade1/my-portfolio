MyHome

MyHome is a real estate mobile app designed to simplify property search, listings, and the home-buying experience. The app puts the entire real estate journey — from browsing to scheduling viewings — into a clean, intuitive mobile experience for iOS and Android.

The Problem

Homebuyers and renters in fast-growing markets struggle with fragmented property listings, unreliable agents, and a lack of transparency — making the search process overwhelming and inefficient.

Strategy

  • Streamlines property discovery with smart filters and map-based search
  • Enables direct communication between buyers and agents within the app
  • Simplifies scheduling property viewings and tours
  • Provides saved searches and push notifications for new listings

Process

  • User research with homebuyers and agents across key markets
  • User flow mapping for the full property search and booking journey
  • Wireframing core mobile screens — search, listings, property detail, scheduling
  • High-fidelity mobile UI with reusable component library

Design Direction

  • Clean mobile-first layout optimized for one-handed browsing
  • Card-based property listings with immersive photo galleries
  • Warm earth tones + blue accent system for a trustworthy, inviting feel

Results

  • Streamlined property discovery with intuitive search and navigation
  • Improved engagement between buyers and real estate agents
  • Built a scalable mobile design system for future feature expansion
